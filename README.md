**VitaLine** is a **prototype web app** for the **Digital Health course**. It shows how secure video calls can help **emergency medical response** by allowing fast visual assessment and coordination **before professionals arrive on-site**.

---

## Features

* Real-time **video calls** between patient and doctor
* **Triage & vitals panel** for quick assessment
* **Emergency escalation** controls (dispatch, alerts)
* **Secure connection** / end-to-end encryption
* Built with **TailwindCSS, Anime.js, ECharts**

---

## Tech

* **Frontend:** HTML5, TailwindCSS, Anime.js, ECharts
* **Backend (planned):** Node.js / WebRTC
* **Deployment:** Static demo on GitHub Pages
