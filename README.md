**VitaLine** is a **prototype web application** developed as part of the **Digital Health university course project**.
Its goal is to demonstrate how secure, real-time video communication can assist **emergency medical response** by enabling rapid visual assessment, triage, and coordination between patients and healthcare professionals **before on-site intervention**.

## üéØ Project Objective

The project explores how video conferencing, geolocation, and live data input (vitals, triage level, connection quality) can improve emergency evaluation and reduce response time.
This is an **academic prototype**, built **for demonstration and research purposes only**, not for real-world medical use.
